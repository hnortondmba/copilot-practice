# copilot-practice

This repository contains a C# console application project created using the .NET template.

## Project Structure

- `CopilotPractice/` - C# console application project
  - `CopilotPractice.csproj` - Project file targeting .NET 8.0
  - `Program.cs` - Main application entry point

## Getting Started

### Prerequisites
- .NET 8.0 or later

### Building the Project
```bash
cd CopilotPractice
dotnet build
```

### Running the Project
```bash
cd CopilotPractice
dotnet run
```

The application will output "Hello, World!" to the console.
